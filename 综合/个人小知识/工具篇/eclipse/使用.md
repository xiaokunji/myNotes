# 二. 不让控制台总是弹出来
`window  ->   preferences   ->  run/debug   ->   console`
在右边面板去掉
```
"Show when program writest to standard out"和
"Show when program writes to standard error"
```
两个多选框，然后重启Eclipse。

 直接选择下图中的按钮,第一个选中时表示正常输出时展现控制台(看英文), 第二个是错误输出时展示
